بنى الشبكات العصبية

تم تصميم بنى شبكات عصبية مختلفة لأنواع مختلفة من البيانات والمشاكل:

CNN

الشبكة العصبية التلافيفية (Convolutional Neural Network)

متخصصة في معالجة البيانات الشبكية مثل الصور. تستخدم طبقات تلافيفية لتعلم الميزات المكانية تلقائياً.

المفهوم الأساسي: المرشحات (النوى) تنزلق عبر الصورة للكشف عن أنماط مثل الحواف والقوام والأشكال.
التطبيقات:
  • تصنيف الصور (قط مقابل كلب)
  • اكتشاف الأجسام (العثور على الوجوه)
  • التصوير الطبي (اكتشاف الأورام)
  • السيارات ذاتية القيادة (التعرف على الطريق)

RNN

الشبكة العصبية المتكررة (Recurrent Neural Network)

مصممة للبيانات التسلسلية حيث يهم الترتيب. لديها "ذاكرة" للمدخلات السابقة من خلال الاتصالات المتكررة.

المفهوم الأساسي: المخرج من خطوة زمنية واحدة يصبح جزءاً من المدخل للخطوة التالية، مما يخلق حلقة تتذكر السياق.
التطبيقات:
  • التنبؤ بالسلاسل الزمنية
  • التنبؤ بأسعار الأسهم
  • التنبؤ بالطقس
  • توليد الموسيقى

LSTM

الذاكرة الطويلة قصيرة المدى (Long Short-Term Memory)

شبكة RNN متقدمة يمكنها تعلم التبعيات طويلة المدى. تحل مشكلة "تلاشي التدرج" في شبكات RNN الأساسية.

المفهوم الأساسي: تستخدم بوابات (نسيان، إدخال، إخراج) للتحكم في المعلومات التي يجب الاحتفاظ بها أو إضافتها أو إخراجها من حالة الخلية.
التطبيقات:
  • التعرف على الكلام
  • الترجمة الآلية
  • توليد النصوص
  • توليد عناوين الفيديو

المحوّل (Transformer)

البنية القائمة على الانتباه (Attention-Based Architecture)

البنية وراء الذكاء الاصطناعي الحديث مثل ChatGPT و BERT و GPT-4. تعالج التسلسلات بالتوازي باستخدام الانتباه الذاتي.

المفهوم الأساسي: الانتباه الذاتي يسمح لكل كلمة بـ"الانتباه" لجميع الكلمات الأخرى، مما يلتقط العلاقات بغض النظر عن المسافة.
التطبيقات:
  • نماذج اللغة الكبيرة (ChatGPT, Claude)
  • الترجمة الآلية (Google Translate)
  • تلخيص النصوص
  • توليد الصور (DALL-E)
التطور: RNN ← LSTM ← Transformer. كل بنية حلت قيود البنية السابقة. المحولات تهيمن الآن على الذكاء الاصطناعي الحديث بسبب التوازي وقدرتها على التقاط التبعيات طويلة المدى.

أول نموذج تعلم آلة لك (scikit-learn)

# Complete example: Classification with scikit-learn from sklearn.datasets import load_iris from sklearn.model_selection import train_test_split from sklearn.ensemble import RandomForestClassifier from sklearn.metrics import accuracy_score # 1. Load data (تحميل البيانات) iris = load_iris() X, y = iris.data, iris.target # 2. Split into train/test (تقسيم إلى تدريب/اختبار) X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2) # 3. Create and train model (إنشاء وتدريب النموذج) model = RandomForestClassifier() model.fit(X_train, y_train) # 4. Make predictions (إجراء التنبؤات) predictions = model.predict(X_test) # 5. Evaluate (التقييم) accuracy = accuracy_score(y_test, predictions) print(f"Accuracy: {accuracy:.2%}")

أول شبكة عصبية لك (Keras)

# Complete example: Neural Network with Keras from tensorflow import keras from tensorflow.keras import layers # 1. Load MNIST dataset (تحميل مجموعة بيانات MNIST) (X_train, y_train), (X_test, y_test) = keras.datasets.mnist.load_data() X_train = X_train.reshape(-1, 784) / 255.0 # Flatten and normalize X_test = X_test.reshape(-1, 784) / 255.0 # 2. Build model (بناء النموذج) model = keras.Sequential([ layers.Dense(128, activation='relu', input_shape=(784,)), layers.Dropout(0.2), layers.Dense(64, activation='relu'), layers.Dense(10, activation='softmax') ]) # 3. Compile (تجميع) model.compile( optimizer='adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'] ) # 4. Train (تدريب) model.fit(X_train, y_train, epochs=5, validation_split=0.1) # 5. Evaluate (تقييم) test_loss, test_acc = model.evaluate(X_test, y_test) print(f"Test accuracy: {test_acc:.2%}")
