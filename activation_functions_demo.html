Why Activation Functions Matter

Activation functions introduce non-linearity into neural networks. Without them, no matter how many layers you stack, the network would just be a linear model. These functions allow neural networks to learn complex patterns like curves, shapes, and abstract features.

Sigmoid

f(x) = 1 / (1 + e^(-x))
Output Range: (0, 1)
Best for: Binary classification output layer. Outputs can be interpreted as probabilities.

ReLU (Rectified Linear Unit)

f(x) = max(0, x)
Output Range: [0, infinity)
Best for: Hidden layers in deep networks. Fast to compute, reduces vanishing gradient problem.

Tanh (Hyperbolic Tangent)

f(x) = (e^x - e^(-x)) / (e^x + e^(-x))
Output Range: (-1, 1)
Best for: Hidden layers when you need zero-centered outputs. Common in RNNs.

Leaky ReLU

f(x) = x if x > 0, else 0.01x
Output Range: (-infinity, infinity)
Best for: Avoiding "dying ReLU" problem where neurons stop learning.
