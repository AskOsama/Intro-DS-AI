مقدمة: ما هي Pandas؟

التعريف: Pandas هي مكتبة بايثون للعمل مع البيانات الجدولية (البيانات في صفوف وأعمدة). الاسم مشتق من "Panel Data" - مصطلح من الاقتصاد القياسي. Pandas تسهل تحميل البيانات ومعالجتها وتحليلها.

الرسالة الأساسية: Pandas هي أساس علم البيانات في بايثون. كما تعلمت استخدام المتغيرات والقوائم في بايثون، ستستخدم إطارات البيانات في Pandas. هذه هي الطريقة التي يعمل بها علماء البيانات مع البيانات الحقيقية.

لماذا نتعلم Pandas؟

  • المعيار الصناعي: يستخدمها علماء البيانات حول العالم
  • تحضير البيانات: 80% من عمل علم البيانات هو تحضير البيانات
  • أداة البوابة: الخطوة الأولى قبل تعلم الآلة
  • تتكامل مع كل شيء: تعمل مع NumPy و Matplotlib و Scikit-learn

# استيراد pandas (الاصطلاح: استخدم 'pd' كاختصار)
import pandas as pd

# التحقق من الإصدار
print(pd.__version__)

١. إطارات البيانات: قلب Pandas

التعريف: إطار البيانات (DataFrame) هو هيكل بيانات ثنائي الأبعاد مُسمى مع أعمدة من أنواع مختلفة محتملة. فكر فيه كجدول بيانات Excel أو جدول SQL في بايثون.

من جدول Excel إلى إطار البيانات

إذا استخدمت Excel من قبل، فأنت تفهم المفهوم الأساسي بالفعل. إطار البيانات مشابه ولكنه أقوى:

المصطلحات الأساسية:

  • الفهرس (Index): تسميات الصفوف (0، 1، 2... افتراضياً) - مثل أرقام الصفوف في Excel
  • الأعمدة (Columns): رؤوس مسماة (الاسم، العمر، الدرجة...) - مثل رؤوس الأعمدة في Excel
  • القيم (Values): البيانات الفعلية في الخلايا
  • نوع البيانات (dtype): نوع بيانات كل عمود (int64, float64, object)

مثال الكود: إنشاء إطار بيانات

import pandas as pd

# إنشاء إطار بيانات من قاموس
data = {
    'Name': ['أحمد', 'سارة', 'علي', 'نورة'],
    'Age': [20, 22, 21, 20],
    'Grade': [85, 92, 78, 95],
    'City': ['الرياض', 'جدة', 'الدمام', 'مكة']
}

df = pd.DataFrame(data)
print(df)

# المخرجات:
#     Name  Age  Grade    City
# 0   أحمد   20     85  الرياض
# 1   سارة   22     92    جدة
# 2    علي   21     78  الدمام
# 3   نورة   20     95    مكة

مهم: لاحظ أن الفهرس يبدأ من 0، وليس 1 مثل أرقام صفوف Excel. هذا فرق أساسي!

٢. السلسلة مقابل إطار البيانات

التعريف: السلسلة (Series) هي مصفوفة مُسماة أحادية البعد. إنها مثل عمود واحد من إطار البيانات، أو قائمة بايثون مع تسميات (فهرس).

فكرة أساسية:

كل عمود في إطار البيانات هو سلسلة. إطار البيانات = مجموعة سلاسل بفهرس مشترك.

مثال الكود: السلسلة مقابل إطار البيانات

import pandas as pd

# إنشاء إطار بيانات
df = pd.DataFrame({
    'Name': ['أحمد', 'سارة', 'علي', 'نورة'],
    'Age': [20, 22, 21, 20],
    'Grade': [85, 92, 78, 95]
})

# استخراج عمود واحد = سلسلة
names = df['Name']
print(type(names))  # <class 'pandas.core.series.Series'>

# استخراج أعمدة متعددة = إطار بيانات
subset = df[['Name', 'Age']]
print(type(subset))  # <class 'pandas.core.frame.DataFrame'>

# إنشاء سلسلة مباشرة
ages = pd.Series([20, 22, 21, 20], name='Age')
print(ages)

تذكر: الأقواس المفردة df['Name'] تُرجع سلسلة. الأقواس المزدوجة df[['Name']] تُرجع إطار بيانات بعمود واحد.

٣. فهم البيانات: ()info مقابل ()describe

التعريف: هاتان طريقتان أساسيتان لاستكشاف بياناتك:

  • info() - تُظهر الهيكل (أنواع البيانات، القيم المفقودة)
  • describe() - تُظهر الإحصائيات (العدد، المتوسط، الانحراف المعياري، الحد الأدنى، الحد الأقصى)

مثال الكود: استكشاف بياناتك

import pandas as pd

df = pd.DataFrame({
    'Name': ['أحمد', 'سارة', 'علي', 'نورة'],
    'Age': [20, 22, 21, 20],
    'Grade': [85.0, 92.0, None, 95.0],  # ملاحظة: None = قيمة مفقودة
    'City': ['الرياض', 'جدة', 'الدمام', 'مكة']
})

# info() - معلومات الهيكل
df.info()
# تُظهر: أسماء الأعمدة، عدد القيم غير الفارغة، أنواع البيانات

# describe() - ملخص إحصائي
df.describe()
# تُظهر: العدد، المتوسط، الانحراف المعياري، الحد الأدنى، 25%، 50%، 75%، الحد الأقصى
# ملاحظة: فقط للأعمدة الرقمية!

متى تستخدم أيهما:

  • info() - أول شيء تنفذه على بيانات جديدة. تجيب: "هل لدي بيانات مفقودة؟ ما هي أنواع البيانات؟"
  • describe() - للتحليل الرقمي. تجيب: "ما المتوسط؟ ما النطاق؟"

٤. الوصول للبيانات: iloc مقابل loc

التعريف: توفر Pandas طريقتين للوصول للبيانات:

  • iloc - integer location - يستخدم أرقام المواقع (0، 1، 2...)
  • loc - label location - يستخدم التسميات (أسماء الصفوف/الأعمدة)

حيلة للتذكر:

  • iloc = "i" للأعداد الصحيحة (Integer)
  • loc = "L" للتسميات (Labels)

مثال الكود: iloc مقابل loc

import pandas as pd

df = pd.DataFrame({
    'Name': ['أحمد', 'سارة', 'علي', 'نورة'],
    'Age': [20, 22, 21, 20],
    'Grade': [85, 92, 78, 95]
})

# iloc - بالموقع (أرقام)
df.iloc[0]           # الصف الأول (موقع 0)
df.iloc[1, 0]        # صف 1، عمود 0 = "سارة"
df.iloc[0:2]         # أول صفين (مواقع 0-1)
df.iloc[:, 0:2]      # كل الصفوف، أول عمودين

# loc - بالتسمية (أسماء)
df.loc[0]            # الصف بتسمية 0
df.loc[1, 'Name']    # تسمية 1، عمود 'Name' = "سارة"
df.loc[:, 'Age']     # كل الصفوف، عمود 'Age'
df.loc[0:2, 'Name':'Age']  # تسميات 0-2، أعمدة Name إلى Age

# الفرق الأساسي: iloc يستثني النهاية، loc يشملها!
df.iloc[0:2]  # صفوف 0، 1 (يستثني 2)
df.loc[0:2]   # صفوف 0، 1، 2 (يشمل 2)

مهم: تقطيع iloc يستثني النهاية (مثل قوائم بايثون)، لكن تقطيع loc يشمل النهاية (لأن التسميات ليس لها قيمة "تالية" طبيعية).

٥. إضافة بيانات جديدة

التعريف: يمكنك توسيع إطار البيانات بإضافة أعمدة أو صفوف جديدة. الأعمدة تُضاف بالتعيين؛ الصفوف تُضاف باستخدام loc أو concat().

مثال الكود: إضافة أعمدة وصفوف

import pandas as pd

df = pd.DataFrame({
    'Name': ['أحمد', 'سارة', 'علي'],
    'Age': [20, 22, 21],
    'Grade': [85, 92, 78]
})

# إضافة عمود جديد
df['Passed'] = df['Grade'] >= 60  # عمود منطقي
df['Grade_Letter'] = ['B', 'A', 'C']  # من قائمة

# إضافة صف جديد باستخدام loc
df.loc[3] = ['نورة', 20, 95, True, 'A']

# إضافة صفوف متعددة باستخدام concat
new_rows = pd.DataFrame({
    'Name': ['خالد', 'فاطمة'],
    'Age': [23, 21],
    'Grade': [88, 91],
    'Passed': [True, True],
    'Grade_Letter': ['B', 'A']
})
df = pd.concat([df, new_rows], ignore_index=True)

print(df)

الأنماط الأساسية:

  • عمود جديد: df['new_col'] = values
  • صف جديد: df.loc[index] = values
  • صفوف متعددة: pd.concat([df1, df2], ignore_index=True)

٦. العمل مع ملفات CSV

التعريف: CSV (قيم مفصولة بفواصل) هو أكثر تنسيق شيوعاً لمشاركة البيانات الجدولية. إنه ملف نصي عادي حيث كل سطر هو صف والقيم مفصولة بفواصل.

مثال الكود: قراءة وكتابة CSV

import pandas as pd

# قراءة ملف CSV
df = pd.read_csv('students.csv')

# المعاملات الشائعة:
df = pd.read_csv('students.csv',
                 encoding='utf-8',      # ترميز الحروف
                 sep=',',               # الفاصل (الفاصلة افتراضياً)
                 header=0,              # الصف المستخدم كرأس
                 index_col=None)        # العمود المستخدم كفهرس

# إلقاء نظرة أولى على بياناتك
print(df.head())    # أول 5 صفوف
print(df.tail())    # آخر 5 صفوف
print(df.shape)     # (صفوف، أعمدة)

# الحفظ إلى CSV
df.to_csv('output.csv', index=False)  # index=False يمنع العمود الإضافي

# لماذا index=False؟
# بدونها تحصل على: ,Name,Age,Grade (عمود إضافي)
# معها تحصل على: Name,Age,Grade (نظيف)

نصيحة: استخدم دائماً index=False عند حفظ CSV إلا إذا كنت تحتاج الفهرس كعمود تحديداً.

٧. معالجة البيانات المفقودة (NaN)

التعريف: NaN (ليس رقماً) هي طريقة Pandas لتمثيل القيم المفقودة. البيانات الحقيقية غالباً بها فجوات - استطلاعات لم تُكمل، أجهزة استشعار فشلت، إلخ.

مثال الكود: اكتشاف ومعالجة NaN

import pandas as pd
import numpy as np

df = pd.DataFrame({
    'Name': ['أحمد', 'سارة', 'علي', 'نورة'],
    'Age': [20, np.nan, 21, 20],
    'Grade': [85, 92, np.nan, 95],
    'City': ['الرياض', 'جدة', 'الدمام', np.nan]
})

# اكتشاف القيم المفقودة
print(df.isnull())       # True حيث يوجد NaN
print(df.isnull().sum()) # عدد NaN لكل عمود

# الاستراتيجية 1: حذف الصفوف مع NaN
df_clean = df.dropna()   # يحتفظ فقط بالصفوف الكاملة

# الاستراتيجية 2: الملء بقيمة محددة
df['Age'].fillna(0, inplace=True)
df['City'].fillna('غير معروف', inplace=True)

# الاستراتيجية 3: الملء بالمتوسط (للأعمدة الرقمية)
mean_grade = df['Grade'].mean()
df['Grade'].fillna(mean_grade, inplace=True)

# أو في سطر واحد:
df['Grade'] = df['Grade'].fillna(df['Grade'].mean())

أي استراتيجية تستخدم:

  • ()dropna: عندما يكون لديك الكثير من البيانات وقليل من القيم المفقودة
  • fillna(value): عندما يكون لديك قيمة افتراضية منطقية
  • fillna(mean()): عندما تريد الحفاظ على الخصائص الإحصائية

٨. التصفية والتحليل الأساسي

التعريف: الفهرسة المنطقية (Boolean Indexing) هي اختيار الصفوف بناءً على شروط. تُنشئ قناع صح/خطأ وتستخدمه لتصفية إطار البيانات.

مثال الكود: تصفية البيانات

import pandas as pd

df = pd.DataFrame({
    'Name': ['أحمد', 'سارة', 'علي', 'نورة', 'خالد'],
    'Age': [20, 22, 21, 20, 23],
    'Grade': [85, 92, 78, 95, 88],
    'City': ['الرياض', 'جدة', 'الدمام', 'مكة', 'الرياض']
})

# تصفية بسيطة: الطلاب بدرجة أعلى من 85
high_achievers = df[df['Grade'] > 85]

# شروط متعددة: استخدم & (و)، | (أو)
riyadh_high = df[(df['City'] == 'الرياض') & (df['Grade'] > 80)]

# استخدام isin() لقيم متعددة
main_cities = df[df['City'].isin(['الرياض', 'جدة'])]

# دوال النصوص
names_with_a = df[df['Name'].str.startswith('أ')]

# إحصائيات ملخصة حسب المجموعة
avg_by_city = df.groupby('City')['Grade'].mean()

# تجميعات متعددة
summary = df.groupby('City').agg({
    'Grade': ['mean', 'max'],
    'Age': 'mean'
})

العمليات الأساسية:

  • df[condition] - تصفية الصفوف
  • & (و)، | (أو) - دمج الشروط
  • groupby() - تجميع البيانات للتجميع
  • agg() - تطبيق دوال متعددة

٩. Pandas في خط أنابيب علم البيانات

الرسالة الأساسية: Pandas ليست أداة معزولة - إنها المحور المركزي الذي يربط البيانات الخام بالتحليل والتصور. فهم موقع Pandas يساعدك على فهم سير عمل علم البيانات بالكامل.

سير عمل علم البيانات النموذجي:

  1. التحميل: ()pd.read_csv - إدخال البيانات إلى بايثون
  2. الاستكشاف: ()df.info(), df.describe(), df.head - فهم ما لديك
  3. التنظيف: ()dropna(), fillna(), astype - إصلاح المشاكل
  4. التحويل: ()groupby(), merge(), apply - إعادة تشكيل البيانات
  5. التحليل: تمرير إطار البيانات النظيف إلى NumPy، Scikit-learn
  6. التصور: تمرير البيانات إلى Matplotlib، Seaborn

مثال الكود: خط أنابيب مصغر كامل

import pandas as pd

# 1. التحميل
df = pd.read_csv('raw_students.csv')

# 2. الاستكشاف
print(df.info())
print(df.describe())

# 3. التنظيف
df = df.dropna()  # إزالة القيم المفقودة
df['Age'] = df['Age'].astype(int)  # إصلاح نوع البيانات

# 4. التحويل
avg_grades = df.groupby('City')['Grade'].mean()

# 5. جاهز للتحليل أو التصور!
print(avg_grades)

١٠. GroupBy: تجميع البيانات حسب المجموعات

المفهوم الأساسي: ()groupby يتيح لك تقسيم بياناتك إلى مجموعات بناءً على قيمة عمود، ثم تطبيق حساب على كل مجموعة، ودمج النتائج. هذا يسمى نمط تقسيم-تطبيق-دمج.

مصطلحات GroupBy

المصطلح الوصف مثال
groupby(column) يجمع الصفوف حسب القيم الفريدة في عمود df.groupby('City')
.mean() حساب المتوسط لكل مجموعة df.groupby('City')['Grade'].mean()
.sum() حساب المجموع لكل مجموعة df.groupby('City')['Sales'].sum()
.count() عد العناصر في كل مجموعة df.groupby('City')['Name'].count()
.min() / .max() إيجاد الحد الأدنى/الأقصى في كل مجموعة df.groupby('City')['Grade'].max()

مثال الكود: عمليات GroupBy

# تجميع بسيط - متوسط الدرجة لكل مدينة
avg_by_city = df.groupby('City')['Grade'].mean()
print(avg_by_city)

# تجميعات متعددة في وقت واحد
summary = df.groupby('City')['Grade'].agg(['mean', 'min', 'max', 'count'])
print(summary)

# التجميع حسب أعمدة متعددة
multi_group = df.groupby(['City', 'Gender'])['Grade'].mean()
print(multi_group)

تذكر: نتيجة groupby() + التجميع لها صف واحد لكل قيمة مجموعة فريدة. إذا قمت بالتجميع حسب المدينة ولديك 3 مدن فريدة، ستكون نتيجتك 3 صفوف.

١١. لغة الاستعلام الطبيعية (NQL) إلى Pandas

التعريف: لغة الاستعلام الطبيعية (NQL) هي نموذج يصف فيه المستخدمون احتياجاتهم لمعالجة البيانات بلغة طبيعية، ثم يترجم نموذج اللغة الكبير (LLM) هذه الطلبات إلى كود قابل للتنفيذ. هذا يسد الفجوة بين نية الإنسان وعمليات pandas.

فكرة أساسية: عندما تتعلم صياغة أسئلة البيانات بوضوح، يمكن لنماذج الذكاء الاصطناعي توليد كود pandas دقيق. الأمثلة أدناه توضح كيفية ربط اللغة الطبيعية بعمليات pandas.

إطار البيانات النموذجي للأمثلة

import pandas as pd

# Create sample DataFrame
df = pd.DataFrame({

print(df)

مثال ١: التصفية البسيطة

مثال ٢: التجميع مع التصنيف

استعلام NQL: "ما هو متوسط الراتب لكل قسم؟"

result = df.groupby('department')['salary'].mean().reset_index()
result.columns = ['department', 'average_salary']
print(result)

الشرح: يربط النموذج "متوسط" بـ mean()، ويحدد مفتاح التصنيف (department)، والعمود المستهدف (salary).

مثال ٣: الترتيب مع اختيار الأعمدة

استعلام NQL: "قائمة الموظفين مرتبة حسب درجة الأداء من الأعلى إلى الأدنى، وأظهر فقط أسماءهم ودرجاتهم"

result = df[['name', 'performance_score']].sort_values(

    ascending=False

الشرح: يستخرج النموذج اختيار الأعمدة، وعمود الترتيب، ويستنتج ascending=False من "من الأعلى إلى الأدنى".

مثال ٤: التصفية المعقدة بشروط متعددة

استعلام NQL: "ابحث عن الموظفين الذين لديهم أكثر من 4 سنوات خبرة وراتب أعلى من 60000"

result = df[(df['years_experience'] > 4) & (df['salary'] > 60000)]

الشرح: يحلل النموذج "و" المنطقية إلى العامل &، مع وضع كل شرط بين أقواس.

مثال ٥: إنشاء أعمدة مشتقة

استعلام NQL: "أضف عموداً جديداً يُظهر الراتب لكل سنة خبرة لكل موظف"

df['salary_per_year_exp'] = df['salary'] / df['years_experience']

مثال ٦: البيانات الوصفية المجهولة (مرجع فهرس العمود)

السياق: للخصوصية، يمكنك إخفاء أسماء الأعمدة من النموذج والإشارة إليها بالموقع بدلاً من ذلك.

استعلام NQL: "احسب مجموع القيم في فهرس العمود 3"

مثال ٧: الملخص الإحصائي

استعلام NQL: "أعطني ملخصاً إحصائياً لعمود الراتب يتضمن العدد والمتوسط والحد الأدنى والحد الأقصى"

result = df['salary'].agg(['count', 'mean', 'min', 'max'])

الشرح: يربط النموذج مصطلحات الإحصاء باللغة الطبيعية مباشرة بطرق التجميع في pandas.

مرجع أنماط NQL

نمط NQL عملية Pandas
"أرني جميع X حيث Y" df[df['column'] == value]
"متوسط/مجموع/عدد X حسب Y" df.groupby('Y')['X'].agg()
"رتب حسب X" df.sort_values(by='X')
"X و Y" (condition1) & (condition2)
"X أو Y" (condition1) | (condition2)

"فهرس العمود N" df.iloc[:, N]

أفضل الممارسات لـ NQL:

  • كن محدداً: "متوسط الراتب" أوضح من "معلومات الراتب"
  • استخدم العوامل الصريحة: "و" و "أو" تساعد النموذج على فهم المنطق
  • اذكر أسماء الأعمدة: عندما لا تكون الخصوصية مصدر قلق، استخدم الأسماء الفعلية
  • استخدم الفهارس للخصوصية: عندما يجب إخفاء البيانات الوصفية، أشر بالموقع
  • حدد تنسيق الإخراج: "أظهر فقط الاسم والراتب" يوجه اختيار الأعمدة

لماذا تشغيل النماذج محلياً؟ عند العمل مع بيانات سرية (سجلات مالية، بيانات طبية، معلومات شخصية)، قد يُخالف إرسال البيانات إلى واجهات برمجة التطبيقات السحابية سياسات الخصوصية. النماذج المحلية تحتفظ بكل شيء على جهازك.

١٣. استخدام مصطلحات Pandas مع نماذج الذكاء الاصطناعي

الربط بالجلسة الثانية: في التطوير بمساعدة الذكاء الاصطناعي، تعلمنا أن المصطلحات الصحيحة ضرورية للتعامل الفعال. مصطلحات Pandas التي تعلمتها اليوم هي بالضبط المفردات التي تفهمها نماذج الذكاء الاصطناعي!

أسئلة جيدة مقابل سيئة:

سؤال سيء (غامض) سؤال جيد (يستخدم المصطلحات)
"كيف أحصل على أول 5 أسطر؟" "كيف أستخدم df.head(5) لعرض أول 5 صفوف من إطار البيانات؟"
"كيف أرى ما في جدولي؟" "كيف أستخدم df.info() للتحقق من أنواع البيانات والقيم المفقودة؟"
"أحصل على الصف الثاني" "كيف أستخدم df.iloc[1] للوصول للصف في الموقع 1؟"
"احذف الخلايا الفارغة" "كيف أستخدم df.dropna() لإزالة الصفوف ذات قيم NaN؟"
"احصل على المتوسط حسب المجموعة" "كيف أستخدم df.groupby('City')['Grade'].mean() لحساب متوسط الدرجة لكل مدينة؟"

مرجع سريع لمصطلحات Pandas للأسئلة:

  • قل "DataFrame" وليس "جدول" أو "ورقة بيانات"
  • قل "Series" وليس "عمود" عند الإشارة للبيانات المستخرجة
  • قل "Index" وليس "رقم الصف"
  • قل "NaN" وليس "فارغ" أو "خالي"
  • قل "iloc" للوصول بالموقع، "loc" للوصول بالتسمية
  • قل "dropna/fillna" وليس "احذف/استبدل الفارغ"

تذكر: المصطلحات التي تعلمتها في هذه الجلسة ليست أكاديمية فقط - إنها المفردات التي تفهمها نماذج الذكاء الاصطناعي. استخدام "DataFrame" بدلاً من "جدول"، و "iloc" بدلاً من "موقع"، و "NaN" بدلاً من "فارغ" سيحصل لك على مساعدة أفضل من الذكاء الاصطناعي.
