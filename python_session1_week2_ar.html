بنهاية هذه الجلسة، سيكون الطلاب قادرين على:

مقدمة: المكونات السبعة لبرمجة بايثون

الرسالة الأساسية: كل برنامج بايثون مبني من نفس اللبنات الأساسية. فهم هذه المكونات السبعة يمنحك الأساس لإنشاء أي برنامج.

فكر في البرمجة مثل البناء بقطع الليغو. كما يمكنك إنشاء أي شيء من قطع ليغو أساسية، يمكنك بناء أي برنامج باستخدام هذه المكونات السبعة الأساسية:

لماذا هذه المكونات مهمة:

  • المتغيرات - تخزن بياناتك
  • نظام الملفات - تحفظ البيانات بشكل دائم
  • جمل التحكم - تتخذ قرارات وتكرر الإجراءات
  • الدوال - تنظم وتعيد استخدام الكود
  • الأصناف والكائنات - تنمذج الأشياء الحقيقية
  • الحزم - تستخدم أدوات جاهزة
  • pip - تثبت أدوات جديدة

١. المتغيرات: حاويات البيانات في الذاكرة

التعريف: المتغير هو حاوية مسماة تخزن البيانات في ذاكرة الحاسوب (RAM). فكر فيه كصندوق مسمى يمكنك وضع المعلومات فيه.

تشبيه الصندوق

تخيل أن لديك صناديق في غرفتك. كل صندوق له تسمية (اسم المتغير) ويحتوي على شيء ما (القيمة). يمكنك:

  • وضع شيء في صندوق (تعيين قيمة)
  • النظر لما بداخله (قراءة القيمة)
  • استبدال ما بداخله (تحديث القيمة)

أنواع البيانات في بايثون

متغيرات بايثون يمكنها حمل أنواع مختلفة من البيانات:

مثال الكود: إنشاء المتغيرات

# النص (String)
name = "أحمد"
city = "الرياض"

# عدد صحيح (Integer)
age = 25
year = 2024

# عدد عشري (Float)
temperature = 36.5
price = 99.99

# منطقي (Boolean)
is_student = True
is_graduated = False

# قائمة (List)
grades = [90, 85, 92, 88]
colors = ["أحمر", "أخضر", "أزرق"]

# قاموس (Dictionary)
student = {
    "name": "أحمد",
    "age": 25,
    "grades": [90, 85, 92]
}

# التحقق من نوع المتغير
print(type(name))    # المخرجات: <class 'str'>
print(type(age))     # المخرجات: <class 'int'>
print(type(grades))  # المخرجات: <class 'list'>

مهم: المتغيرات توجد فقط في الذاكرة (RAM). عندما ينتهي برنامجك أو يُغلق حاسوبك، البيانات في المتغيرات تُفقد! لذلك نحتاج نظام الملفات للتخزين الدائم.

٢. نظام الملفات: التخزين الدائم

التعريف: نظام الملفات هو المكان الذي تُخزن فيه البيانات بشكل دائم على القرص الصلب أو SSD. على عكس المتغيرات في الذاكرة، البيانات المحفوظة في الملفات تبقى حتى بعد انتهاء البرنامج.

الذاكرة مقابل التخزين: فهم الفرق

متى تستخدم ماذا:

  • المتغيرات (RAM): للبيانات التي تعمل عليها أثناء تنفيذ البرنامج
  • الملفات (التخزين): للبيانات التي تحتاج الاحتفاظ بها بعد انتهاء البرنامج، أو مشاركتها، أو استخدامها لاحقاً

مثال الكود: العمل مع الملفات

# كتابة البيانات إلى ملف
data = "معلومات مهمة أريد حفظها"

with open("my_data.txt", "w", encoding="utf-8") as file:
    file.write(data)
# الملف يُغلق تلقائياً بعد كتلة 'with'

# قراءة البيانات من ملف
with open("my_data.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)  # المخرجات: معلومات مهمة أريد حفظها

# الإضافة إلى ملف (إضافة بدون الكتابة فوق)
with open("my_data.txt", "a", encoding="utf-8") as file:
    file.write("\nبيانات إضافية أضيفت لاحقاً")

# العمل مع عدة أسطر
grades = [90, 85, 92, 88]
with open("grades.txt", "w") as file:
    for grade in grades:
        file.write(str(grade) + "\n")

تشبيه من الواقع:

فكر فيها هكذا:

  • الذاكرة (المتغيرات) = مكتبك حيث تعمل بنشاط
  • التخزين (الملفات) = خزانة الملفات حيث تحفظ الوثائق

تأخذ الملفات من الخزانة إلى مكتبك للعمل عليها، ثم تعيدها عند الانتهاء.

٣. جمل التحكم: توجيه تدفق البرنامج

التعريف: جمل التحكم تسمح لك بتغيير تدفق برنامجك. بدلاً من تشغيل كل سطر من الأعلى للأسفل، يمكنك اتخاذ قرارات (التفرع) أو تكرار الإجراءات (الحلقات).

تصور تدفق البرنامج

If/Else: اتخاذ القرارات

استخدم if/else عندما تحتاج أن يتخذ برنامجك خيارات بناءً على شروط:

مثال الكود: جمل If/Else

# جملة if بسيطة
temperature = 35

if temperature > 30:
    print("الجو حار اليوم!")

# جملة if/else
age = 18

if age >= 18:
    print("يمكنك التصويت")
else:
    print("لا يمكنك التصويت بعد")

# if/elif/else لشروط متعددة
grade = 85

if grade >= 90:
    print("ممتاز! التقدير: A")
elif grade >= 80:
    print("جيد جداً! التقدير: B")
elif grade >= 70:
    print("جيد! التقدير: C")
else:
    print("يحتاج تحسين")

حلقة For: التكرار عدد معروف من المرات

استخدم حلقات for عندما تعرف كم مرة تريد التكرار:

مثال الكود: حلقات For

# التكرار خلال قائمة
grades = [90, 85, 92, 88]

for grade in grades:
    print(f"الدرجة: {grade}")

# المخرجات:
# الدرجة: 90
# الدرجة: 85
# الدرجة: 92
# الدرجة: 88

# التكرار باستخدام range
for i in range(5):
    print(f"العدد: {i}")

# المخرجات: العدد: 0, 1, 2, 3, 4

# حساب المجموع باستخدام حلقة
total = 0
for grade in grades:
    total = total + grade

average = total / len(grades)
print(f"المتوسط: {average}")  # المخرجات: المتوسط: 88.75

حلقة While: التكرار حتى يتغير شرط

استخدم حلقات while عندما لا تعرف كم مرة تريد التكرار:

مثال الكود: حلقات While

# العد التنازلي
count = 5

while count > 0:
    print(count)
    count = count - 1

print("انطلاق!")

# المخرجات: 5, 4, 3, 2, 1, انطلاق!

# الاستمرار بالسؤال حتى إدخال صحيح
password = ""

while password != "secret123":
    password = input("أدخل كلمة المرور: ")

print("تم منح الوصول!")

تذكر: جمل التحكم هي كيف تعطي برنامجك "ذكاء" - القدرة على اتخاذ قرارات والاستجابة لمواقف مختلفة.

٤. الدوال: كتل الكود القابلة لإعادة الاستخدام

التعريف: الدالة هي كتلة كود مسماة تؤدي مهمة محددة. تأخذ مدخلات (معاملات)، تعالجها باستخدام جمل التحكم، وترجع مخرجات اختيارياً.

صيغة الدالة: مدخلات ← معالجة ← مخرجات

لماذا نستخدم الدوال؟

  • إعادة الاستخدام: اكتب الكود مرة واحدة، استخدمه مرات عديدة
  • التنظيم: قسّم المشاكل المعقدة إلى قطع أصغر
  • القراءة: أعطِ أسماء ذات معنى لكتل الكود
  • الاختبار: اختبر القطع الصغيرة بشكل مستقل

مثال الكود: إنشاء واستخدام الدوال

# تعريف دالة بسيطة
def greet(name):
    """هذه الدالة تحيّي شخصاً"""
    return f"مرحباً، {name}!"

# استدعاء الدالة
message = greet("أحمد")
print(message)  # المخرجات: مرحباً، أحمد!

# دالة بمعاملات متعددة
def calculate_average(grades):
    """حساب متوسط قائمة من الدرجات"""
    total = sum(grades)
    count = len(grades)
    average = total / count
    return average

# استخدام الدالة
my_grades = [90, 85, 92, 88]
result = calculate_average(my_grades)
print(f"المتوسط: {result}")  # المخرجات: المتوسط: 88.75

# دالة بمعامل افتراضي
def power(base, exponent=2):
    """حساب الأساس مرفوعاً للأس"""
    return base ** exponent

print(power(5))      # المخرجات: 25 (تستخدم الافتراضي exponent=2)
print(power(5, 3))   # المخرجات: 125 (5^3)

# دالة لا ترجع شيئاً
def print_report(student_name, grades):
    """طباعة تقرير درجات منسق"""
    print(f"=== تقرير {student_name} ===")
    for i, grade in enumerate(grades, 1):
        print(f"  المادة {i}: {grade}")
    print(f"  المتوسط: {calculate_average(grades):.2f}")

print_report("سارة", [95, 88, 92])

أفضل الممارسات للدوال:

  • أعطِ الدوال أسماء وصفية توضح ما تفعله
  • كل دالة يجب أن تفعل شيئاً واحداً بشكل جيد
  • استخدم المعاملات للإدخال، و return للإخراج
  • أضف docstrings لشرح ما تفعله الدالة

٥. الأصناف والكائنات: تنظيم البيانات والسلوك

التعريف: الصنف هو قالب يحدد كلاً من البيانات (السمات) والسلوك (الدوال) معاً. الكائن هو نسخة محددة أُنشئت من ذلك القالب.

تشبيه المخطط

فكر في الصنف كمخطط معماري لمنزل:

  • المخطط (الصنف) يحدد ما يجب أن يحتويه المنزل: غرف، أبواب، نوافذ
  • كل منزل فعلي (كائن) يُبنى من ذلك المخطط بتفاصيل محددة
  • يمكنك بناء عدة منازل مختلفة من نفس المخطط

مثال الكود: إنشاء صنف

# تعريف صنف (القالب) - Define a class (template)
class طالب:
    """صنف يمثل طالباً - A class representing a student"""

    def __init__(self, الاسم, العمر):
        """تهيئة طالب جديد بالاسم والعمر"""
        self.الاسم = الاسم        # سمة: اسم الطالب
        self.العمر = العمر        # سمة: عمر الطالب
        self.الدرجات = []         # سمة: قائمة الدرجات

    def أضف_درجة(self, درجة):
        """إضافة درجة لسجل الطالب"""
        self.الدرجات.append(درجة)

    def احصل_على_المتوسط(self):
        """حساب وإرجاع متوسط الدرجات"""
        if len(self.الدرجات) == 0:
            return 0
        return sum(self.الدرجات) / len(self.الدرجات)

    def اعرض_المعلومات(self):
        """عرض معلومات الطالب"""
        print(f"الاسم: {self.الاسم}")
        print(f"العمر: {self.العمر}")
        print(f"الدرجات: {self.الدرجات}")
        print(f"المتوسط: {self.احصل_على_المتوسط():.2f}")


# إنشاء كائنات (نسخ) من الصنف - Create objects from the class
طالب1 = طالب("أحمد", 20)
طالب2 = طالب("سارة", 22)

# استخدام دوال الكائن - Using object methods
طالب1.أضف_درجة(90)
طالب1.أضف_درجة(85)
طالب1.أضف_درجة(92)

طالب2.أضف_درجة(95)
طالب2.أضف_درجة(88)

# الوصول لسمات ودوال الكائن
print(طالب1.الاسم)              # المخرجات: أحمد
print(طالب1.احصل_على_المتوسط()) # المخرجات: 89.0

طالب1.اعرض_المعلومات()
# المخرجات:
# الاسم: أحمد
# العمر: 20
# الدرجات: [90, 85, 92]
# المتوسط: 89.00

الفكرة الأساسية: الأصناف تجمع البيانات (ما يعرفه الكائن) والدوال (ما يمكن للكائن فعله) معاً. هذا يجعل الكود أكثر تنظيماً ويعكس طريقة تفكيرنا في الأشياء الحقيقية.

متى تستخدم الأصناف:

  • عندما يكون لديك بيانات تنتمي طبيعياً معاً (مثل اسم الطالب، العمر، الدرجات)
  • عندما تحتاج لإنشاء عدة أشياء متشابهة (مثل عدة طلاب)
  • عندما ترتبط البيانات والعمليات عليها ارتباطاً وثيقاً

٦. الحزم والوحدات والمكتبات

التعريف: الحزمة/المكتبة هي مجموعة من الكود المكتوب مسبقاً (دوال، أصناف، إلخ) يمكنك استيرادها واستخدامها في برامجك. هذا يوفر عليك "إعادة اختراع العجلة".

فهم التسلسل الهرمي

  • الوحدة: ملف بايثون واحد (.py) يحتوي على دوال وأصناف
  • الحزمة: مجموعة من الوحدات المرتبطة منظمة في مجلد
  • المكتبة: مجموعة من الحزم (غالباً تُستخدم بالتبادل مع "حزمة")

NumPy: مثال عملي

NumPy هي واحدة من أهم مكتبات بايثون لعلم البيانات. توفر أدوات قوية للعمل مع الأرقام والمصفوفات.

مثال الكود: استخدام NumPy

# استيراد مكتبة numpy
import numpy as np

# إنشاء مصفوفة (أقوى من القائمة)
grades = np.array([90, 85, 92, 88, 95])

# NumPy توفر دوالاً مفيدة كثيرة
print(f"المصفوفة: {grades}")
print(f"المتوسط: {np.mean(grades)}")       # المتوسط
print(f"الأعلى: {np.max(grades)}")         # القيمة الأعلى
print(f"الأدنى: {np.min(grades)}")         # القيمة الأدنى
print(f"المجموع: {np.sum(grades)}")         # المجموع
print(f"الانحراف المعياري: {np.std(grades):.2f}") # الانحراف المعياري

# المخرجات:
# المصفوفة: [90 85 92 88 95]
# المتوسط: 90.0
# الأعلى: 95
# الأدنى: 85
# المجموع: 450
# الانحراف المعياري: 3.41

# عمليات المصفوفات سهلة
doubled = grades * 2
print(f"المضاعفة: {doubled}")  # [180 170 184 176 190]

# التصفية بالشروط
passing = grades[grades >= 90]
print(f"الدرجات >= 90: {passing}")  # [90 92 95]

# إنشاء مصفوفات بسهولة
zeros = np.zeros(5)         # [0. 0. 0. 0. 0.]
ones = np.ones(3)           # [1. 1. 1.]
sequence = np.arange(0, 10, 2)  # [0 2 4 6 8]

أنماط الاستيراد الشائعة:

٧. تثبيت الحزم باستخدام pip

التعريف: pip هو مدير حزم بايثون. يحمّل الحزم من فهرس حزم بايثون (PyPI) ويثبتها على حاسوبك، يتعامل مع جميع المتطلبات تلقائياً.

كيف يعمل pip

عندما تشغّل pip install numpy، يقوم pip بـ:

  1. الاتصال بـ PyPI (فهرس حزم بايثون)
  2. تحميل حزمة numpy
  3. تحميل أي حزم تعتمد عليها numpy
  4. تثبيت كل شيء في المكان الصحيح

أوامر pip الشائعة

# تثبيت حزمة
pip install numpy

# تثبيت إصدار محدد
pip install numpy==1.24.0

# تثبيت عدة حزم
pip install numpy pandas matplotlib

# ترقية حزمة
pip install --upgrade numpy

# عرض الحزم المثبتة
pip list

# عرض معلومات عن حزمة
pip show numpy

# إزالة حزمة
pip uninstall numpy

# التثبيت من ملف متطلبات
pip install -r requirements.txt

للبدء: لتثبيت حزم بايثون على حاسوبك، افتح الطرفية (Command Prompt في ويندوز، Terminal في ماك/لينكس) واكتب أوامر pip.

الحزم الشائعة لعلم البيانات

الحزم الأساسية للتثبيت:

# حزم علم البيانات الأساسية
pip install numpy          # الحوسبة العددية
pip install pandas         # تحليل البيانات
pip install matplotlib     # التصور البياني
pip install seaborn        # التصور الإحصائي

# تعلم الآلة
pip install scikit-learn   # خوارزميات تعلم الآلة

# دفاتر جوبيتر (البرمجة التفاعلية)
pip install jupyter notebook

أفضل الممارسات:

  • دائماً تحقق إذا كانت الحزمة مثبتة بـ pip list
  • استخدم البيئات الافتراضية للمشاريع المختلفة
  • احتفظ بملف requirements.txt يسرد متطلبات مشروعك

الملخص والخطوات التالية

ما تعلمناه اليوم:

غطينا اللبنات السبعة الأساسية لبرمجة بايثون:

  5. احفظ النتائج في ملف
  6. استخدم NumPy للحسابات

معاينة الجلسة القادمة:

في الجلسة القادمة، سنستكشف كيفية استخدام الذكاء الاصطناعي والنماذج اللغوية الكبيرة (LLMs) مثل ChatGPT و Claude لمساعدتك في كتابة الكود. ستتعلم كيف:

  • تطلب من الذكاء الاصطناعي توليد كود بايثون لمهام محددة
  • تصحيح وإصلاح الأخطاء بمساعدة الذكاء الاصطناعي
  • فهم وشرح الكود الموجود باستخدام النماذج اللغوية
  • أفضل الممارسات للتواصل مع الذكاء الاصطناعي للمساعدة البرمجية
